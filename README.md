# Is modules good?
Yes, its purely amazing. I haven't made a big enough project to talk out of experience but modules also decrease compilation time. Whilst it doesn't have much of a runtime performance increase from my [quick google search](https://letmegooglethat.com/?q=does+c%2B%2B+modules+improve+runtime+performance)
it does help a ton in keeping code clean and avoiding #include messes.
